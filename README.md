## Rodando o programa

Lembre-se de alterar a distacia e a quantidade de cavalos(threads) para as suas especificações.
Execute o Makefile para gerar os objetos de compilação e então o arquivo executável:



```bash
make
```

Execute o arquivo final:

```bash
./jogar
```
